현재 `src/views/viewing/ViewingListView.vue`는 실제 API 데이터 대신 샘플 데이터(`v-for="n in 4"`)를 사용하여 목록을 표시하고 있습니다. 각 관람 항목의 사진을 클릭하여 상세 보기 화면으로 이동하려면, 각 관람 항목에 고유한 `id`가 필요합니다. 하지만 현재 샘플 데이터에서는 이 `id`를 사용할 수 없습니다.

두 가지 방법이 있습니다:
1.  **임시 ID 사용:** `n`을 임시 `id`로 사용하여 상세 보기 화면으로 이동하도록 구현할 수 있습니다. 이렇게 하면 내비게이션은 작동하지만, 상세 보기 화면에는 의미 있는 데이터가 표시되지 않을 수 있습니다.
2.  **API 통합 후 구현:** 먼저 `ViewingListView.vue`가 실제 관람 데이터를 가져오도록 API를 통합한 후, 고유 `id`를 사용하여 상세 보기 링크를 구현할 수 있습니다.
